## emacs 快捷键功能


### 文件操作
ctrl+x ctrl+f  快速打开文件
ctrl+x ctrl+s  保存文件
ctrl+x ctrl+c  退出emacs

### buffer管理
ctrl+x b  切换buffer

### 跳转移动
alt+n 向下移动10行
alt+p 向上移动10行
home   移动到行首
end    移动到行末


### 搜索
ctrl+j ctrl+space  输入单词或者字母 选择位置字母 回车 快速跳转 类似vim的easymotion
ctrl+s 搜索关键词




### 其他
F3  高亮光标处的单词
